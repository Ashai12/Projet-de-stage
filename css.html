Qu'est-ce que CSS ?

CSS signifie Cascading Style Sheets, qu’on peut traduire par Feuilles de style en cascade.

Si HTML est la structure d’un site web (comme les murs et les fondations d’une maison), alors CSS, c’est la peinture, la décoration et l’aménagement intérieur !

Avec CSS, on peut :


  • Changer les couleurs du texte et du fond
  • Modifier la taille et le style des polices
  • Organiser les éléments (les aligner, les espacer, etc.)
  • Ajouter des animations et des effets sympas

En résumé, CSS permet de donner du style aux pages web !

Qui a créé CSS ?

CSS a été inventé par Håkon Wium Lie, un informaticien norvégien.

En 1994, alors qu’il travaillait avec Tim Berners-Lee (l’inventeur du web), il s’est rendu compte qu’HTML seul ne suffisait pas pour faire des sites web jolis et bien organisés. Il a donc proposé un nouveau langage : CSS !

Depuis, CSS a beaucoup évolué, et aujourd’hui, on utilise principalement CSS3, qui permet d’ajouter des effets avancés comme des animations, des dégradés et des ombres.

Comment fonctionne CSS ?

CSS fonctionne avec des règles de style qui ciblent des éléments HTML. Une règle CSS est composée de :

  1. Un sélecteur (l’élément HTML à styliser)
  2. Des propriétés (ce qu'on veut modifier)
  3. Des valeurs (comment on veut modifier)

Exemple :
p {
color: blue; /* Le texte devient bleu */
font-size: 20px; /* La taille du texte est de 20 pixels */
}

Ici, on dit que tous les <p> (paragraphes) doivent avoir :

  • Une couleur bleue
  • Une taille de 20 pixels

Comment utilise-t-on CSS ?

Il y a 3 façons d’ajouter du CSS à une page web :

  1. Dans une balise HTML (peu recommandé) : <p style="color: red;">Texte en rouge</p></li>
  2. Dans une balise <style> dans le fichier HTML :
    <style>
    p color: green;
    </style>
  3. Dans un fichier CSS séparé (la meilleure pratique) :
    body {
    background-color: lightgray;
    }
    Puis on le relie dans HTML avec : <link rel="stylesheet" href="styles.css">

Pourquoi apprendre CSS ?

Si tu veux créer un site web, CSS est indispensable !
Même si HTML fonctionne sans CSS, sans lui, une page serait juste du texte noir sur fond blanc, sans mise en forme.

Avec CSS, on peut créer des designs modernes, adaptés à tous les écrans (ordinateurs, tablettes, mobiles). Et en plus, c’est intéressant à apprendre !
