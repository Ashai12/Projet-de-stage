Qu'est-ce que HTML ?

HTML signifie HyperText Markup Language, ce qui peut paraître compliqué, mais en gros, c'est un langage qui permet de structurer une page web. Grâce à HTML, on peut dire à un navigateur (comme Google Chrome ou Firefox) :


  • "Voici un titre"
  • "Ici, mets un paragraphe"
  • "Ajoute une image à cet endroit"
  • "Crée un lien vers une autre page"

En bref, c'est un peu comme si on écrivait un plan détaillé d'une page web.

Qui a créé HTML ?

HTML a été inventé par Tim Berners-Lee en 1991. À l'époque, il travaillait au CERN, un grand centre de recherche scientifique en Suisse.
Son but était de permettre aux scientifiques du monde entier de partager facilement des documents et des informations. Il a donc créé le tout premier site web et, avec lui, le premier langage HTML ! Depuis, HTML a beaucoup évolué. Aujourd'hui, nous en sommes à HTML5, qui permet de faire des pages web encore plus interactives et modernes.

HTML seul, ça suffit ?

Non, HTML donne la structure, mais pour rendre un site plus joli et interactif, on utilise souvent d'autres langages en complément :

  • CSS (Cascading Style Sheets) : pour la mise en page, les couleurs, les animations...
  • JavaScript : pour ajouter de l'interactivité (ex : un menu qui s'ouvre, un formulaire dynamique, etc.).

Sans CSS et JavaScript, une page en pur HTML serait assez basique, un peu comme un document Word sans mise en page.

Pourquoi apprendre HTML ?

Si tu veux créer un site web, HTML est la base ! Même si tu ne veux pas devenir développeur, comprendre HTML peut être utile, que ce soit pour modifier un blog, optimiser un site, ou même comprendre comment fonctionne le web.

Si ça t'intéresse, tu peux commencer facilement en testant quelques lignes de code dans ton navigateur avec la fonction Inspecter l’élément.
