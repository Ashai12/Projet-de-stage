Qu'est-ce que JavaScript ?

JavaScript (souvent abrégé JS) est un langage de programmation qui permet de rendre une page web interactive. Sans lui, un site serait statique : du texte, des images, des liens… mais rien ne bouge.

Avec JavaScript, on peut :

  • Ajouter des animations
  • Créer des menus interactifs
  • Valider des formulaires (comme quand tu oublies un champ et que ça affiche un message)
  • Mettre à jour du contenu sans recharger la page (comme les notifications sur Facebook)
  • Créer des jeux ou des applications web

Qui a créé JavaScript ?

En 1995, un ingénieur appelé Brendan Eich, travaillant chez Netscape, devait inventer un langage pour les pages web... en 10 jours seulement !

Résultat : il crée JavaScript, un langage inspiré de plusieurs autres langages (comme Java et C).

Au départ, personne ne prenait JavaScript au sérieux... mais aujourd’hui, c’est l’un des langages les plus utilisés dans le monde !

Comment utilise-t-on JavaScript ?

Il y a 3 façons d’ajouter du JavaScript dans une page web :

  1. Dans une balise HTML (peu recommandé) : <button onclick="alert("Salut !")">Clique ici </button>
  2. Dans une balise <script> dans le fichier HTML :
    <script>
    console.log("Ceci s'affiche dans la console !")
    </script>
  3. Dans un fichier .js séparé (la meilleure pratique) :
    document.body.style.backgroundColor = "lightblue";
    Puis on le relie dans HTML avec : <script src="script.js"> </script>

Pourquoi apprendre JavaScript ?

JavaScript est partout ! Il est utilisé par 95% des sites web (Google, YouTube, Facebook, Amazon…).

C’est un des langages les plus demandés en entreprise, parfait pour une carrière de développeur.

Il permet de tout faire : sites web, applications mobiles, jeux vidéo, intelligence artificielle, etc.
