Qu'est-ce que SQL ?

SQL signifie "Structured Query Language", soit "Langage de requêtes structuré" en français.

C'est un langage de programmation conçu pour interagir avec des bases de données. Contrairement à HTML, CSS ou JavaScript, qui s'occupent de l'affichage d'un site web, SQL permet de stocker et manipuler des données en arrière-plan.

Qui a créé SQL ?

SQL a été inventé dans les années 1970 par Donald D. Chamberlin et Raymond F. Boyce, deux chercheurs d'IBM. Ils travaillaient sur un projet appelé System R, l'un des premiers systèmes de bases de données relationnelles.

Le langage s’appelait d’abord SEQUEL, mais pour des raisons légales, il a été renommé SQL.

En 1986, SQL est devenu un standard international, ce qui signifie qu’il est utilisé dans le monde entier et sur différents systèmes comme MySQL, PostgreSQL, SQL Server, Oracle, etc.

Pourquoi SQL est-il important ?

  • SQL est utilisé partout ! Les réseaux sociaux, les e-commerces, les applications mobiles, les banques, les hôpitaux… Tous stockent des données dans des bases SQL.
  • Il permet de gérer d'énormes quantités de données. Google, Amazon, Facebook et presque toutes les entreprises l’utilisent !
  • Il est facile à apprendre et ultra puissant. En quelques lignes de code, tu peux récupérer des milliers d'informations en une fraction de seconde.

Que peut-on faire avec SQL ?

  • Créer des bases de données
  • Ajouter des données (insertion)
  • Lire des données (recherche)
  • Modifier des données
  • Supprimer des données
  • Sécuriser l'accès aux informations

Comment fonctionne SQL ?

SQL fonctionne avec des requêtes : ce sont des commandes qu'on envoie à une base de données.

Créer une table pour stocker des utilisateurs :

CREATE TABLE utilisateurs (
id INT PRIMARY KEY AUTO_INCREMENT,
nom VARCHAR(50),
email VARCHAR(100)
);

Cela crée une table utilisateurs avec trois colonnes : id, nom et email.


Ajouter un utilisateur :

INSERT INTO utilisateurs (nom, email)
VALUES ('Alice Dupont', 'alice@example.com');

Un nouvel utilisateur est ajouté à la base de données.


Mettre à jour un utilisateur :

UPDATE utilisateurs
SET email = 'nouveau@email.com'
WHERE id = 1;

Change l’email de l’utilisateur ayant id = 1.


Supprimer un utilisateur :

DELETE FROM utilisateurs WHERE id = 1;

Supprime l'utilisateur avec id = 1.

Les bases de données SQL populaires

Il existe plusieurs systèmes de gestion de bases de données (SGBD) qui utilisent SQL :

  • MySQL → Très utilisé pour les sites web (WordPress, Magento).
  • PostgreSQL → Puissant et open-source.
  • SQL Server → Développé par Microsoft, souvent utilisé en entreprise.
  • Oracle Database → Très performant pour les grandes entreprises.
  • SQLite → Léger et simple, parfait pour les applications mobiles.

Pourquoi apprendre SQL ?

  • C'est facile à apprendre et fonctionne avec tous les langages de programmation (PHP, Python, Java, etc.).
  • Très demandé en entreprise, car toutes les applications modernes utilisent des bases de données.
  • Puissant et rapide : en quelques requêtes, tu peux manipuler des millions de lignes de données.
  • Indispensable pour le développement web, l’analyse de données et la cybersécurité.
