Qu'est-ce que PHP ?

PHP signifie "Hypertext Preprocessor", même si à l'origine, c'était "Personal Home Page".

Contrairement à HTML et JavaScript, qui fonctionnent directement dans ton navigateur, PHP s’exécute sur un serveur. Il sert principalement à gérer des données et créer des pages dynamiques.

En gros, PHP permet à un site de fonctionner comme une vraie application, avec des bases de données et des interactions plus poussées !

Qui a créé PHP ?

PHP a été créé en 1994 par Rasmus Lerdorf, un développeur d’origine danoise-canadienne.

À la base, il avait développé un petit script en C pour gérer son site personnel. Puis, il l’a amélioré pour permettre aux développeurs de créer des sites dynamiques. Il ne pensait pas que PHP deviendrait l’un des langages les plus utilisés au monde !

À quoi sert PHP ?

PHP est utilisé principalement pour les sites dynamiques, où les données changent tout le temps.

Exemples d'utilisation de PHP :

  • Gérer des utilisateurs (création de comptes, connexion)
  • Traiter des formulaires (enregistrer des informations, envoyer des emails)
  • Gérer des bases de données (stocker des articles de blog, des produits, etc.)
  • Créer des sites e-commerce
  • Créer des forums et des blogs
  • Construire des API et des applications web

Comment fonctionne PHP ?

PHP fonctionne côté serveur. Contrairement à JavaScript, qui est exécuté par ton navigateur, PHP s’exécute sur un serveur web, puis envoie un résultat en HTML que le navigateur peut afficher.

Exemple :

<?php
echo "Bonjour, bienvenue sur mon site !";
?>

Cela affiche simplement :

Bonjour, bienvenue sur mon site !

Tu remarques que PHP commence toujours par <?php et termine par ?> !

PHP et les bases de données

L’un des grands avantages de PHP, c’est qu’il peut se connecter à des bases de données comme MySQL pour stocker et récupérer des informations.

Exemple : Récupérer les utilisateurs d’une base de données

<?php
$connexion = mysqli_connect("localhost", "root", "", "ma_base");

$resultat = mysqli_query($connexion, "SELECT * FROM utilisateurs");

while ($utilisateur = mysqli_fetch_assoc($resultat)) {
echo $utilisateur["nom"] . "
";
}
?>

Ici, PHP récupère tous les noms des utilisateurs enregistrés dans une base de données !

Pourquoi apprendre PHP ?

  • PHP est utilisé par plus de 75% des sites web qui ont un backend (serveur) !
  • Il permet de créer des sites dynamiques et professionnels.
  • Compatible avec presque tous les hébergements web, et souvent préinstallé.
